# SQL Sales Data Analysis using MySQL

## Objective
Analyze retail sales data using SQL to identify business insights and sales trends.

## Tools Used
- SQL
- MySQL Workbench

## Analysis Performed
- Total sales analysis
- Sales by category
- Sales by region
- Top-performing sub-categories

## Key Insights
- Technology generated highest sales
- West region contributed maximum revenue
- Phones became top-performing sub-category

## Skills Demonstrated
- SQL Queries
- GROUP BY
- ORDER BY
- Aggregate Functions
- Data Analysis
